# Cybersecurity Portfolio

This repository contains cybersecurity tools, KQL queries, and threat hunting resources developed for learning, automation, and operational use.

## Tools

### IOC-Reputation-Checker

PowerShell-based widget designed to simplify IOC reputation analysis by integrating VirusTotal and AbuseIPDB integrations.

Supported indicators:

* IPv4
* IPv6
* Domains
* URLs
* File Hashes (MD5, SHA1, SHA256)

Main features:

* Automatic IOC type detection
* VirusTotal integration
* AbuseIPDB integration
* Input validation and sanitization
* Lightweight PowerShell WPF interface

## KQL Queries

A collection of Microsoft Sentinel and Defender hunting queries focused on:

* Sign-in analysis
* Threat hunting
* Authentication investigations
* Endpoint telemetry analysis

Current examples:

* SigninLogs
* AADNonInteractiveUserSignInLogs

## Technologies

* Microsoft Sentinel
* Microsoft Defender XDR
* Kusto Query Language (KQL)
* PowerShell
* WPF
